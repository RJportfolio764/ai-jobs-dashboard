The problem

Why I built this

The old way
Building a Power BI dashboard from scratch takes 6-8 hours. You manually choose KPIs, design layouts, pick colors, write DAX, format charts, and check numbers. Each step requires context-switching between tools.
My approach
A 3-agent AI pipeline where each agent has one job. Agent 1 designs the spec. Agent 2 generates the code and formulas. Agent 3 verifies every number. Same quality dashboard in under 2 hours, with built-in QA.
The system

How the pipeline works

Why three separate agents?
Long AI conversations cause context drift — the model starts averaging across conflicting instructions. Each agent has a focused system prompt, isolated context, and one job. The system prompt is the reusable asset; conversations are disposable. This is how production AI workflows are designed.
